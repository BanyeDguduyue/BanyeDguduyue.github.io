apply,call和bind

这两个方法都是用来改变this指向,call是这样的call(obj,a,b,c,d)，apply是这样的apply(obj,[a,b,c]),apply参数是对象和数组，而call的参数可以写成多个。
看看实际例子

const obj1 = {
  age: '66',
  name: 'lee',
  fun(a, b, c) {
    console.log(`我是${this.name}，今年${this.age}，我白天${a},晚上${b},方法${c}`);
  }
}
const obj = {
  name: "半夜的孤独月",
  age: '22'
};
const val1 = '敲代码',
val2 = '健身'
obj1.fun.call(obj, val1, val2, 'call');
obj1.fun.apply(obj, [val1, val2, 'apply']);
obj1.fun.bind(obj, val1, val2, 'bind')();
obj1.fun.bind(obj, [val1, val2, 'apply'])();


若传入的参数不对就会像第四条打印的那样，相当于传入了第一个参数后两个是undefined

从图片中可以看出：

  • call的第二个参数是单个的。
  • apply的第二个参数是一个数组。
  • bind的和call类似，但是返回的是一个方法需要执行

三者的参数不限定是 string 类型，允许是各种类型，包括函数、object
