何为闭包

闭包可以简单理解成“定义在一个函数内部的函数“。能够取得函数的变量值。

  function func1(){
    var a = 0
    return function func2(){
      a++
      console.log(a)
    }
  }

  var b = func1()
  b()
  b()


由图可以看出每执行一次就会让a+1，在var b = func1()的时候相当于执行了func1() b => func2 变量a存在于内存中 以后每次执行b都会使得内存中定义的a+1

闭包的三个特点

*函数嵌套函数。
*函数内部可以引用外部的参数和变量。
*参数和变量不会被垃圾回收机制回收。

闭包的好处

*希望一个变量长期存储在内存中。
*避免全局变量的污染。
*私有成员的存在。

闭包的缺点

*常驻内存，增加内存使用量。
*使用不当会很容易造成内存泄露。
